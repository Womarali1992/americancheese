# Site Setups Landing Page

Landing page for [sitesetups.com](https://sitesetups.com) - AI Context Engineering & Project Management.

## Development

```bash
npm install
npm run dev
```

## Build & Deploy

```bash
npm run build
scp -r dist/* root@134.199.207.43:/var/www/landing/
```

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Radix UI / shadcn-ui
